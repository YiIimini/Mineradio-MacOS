# MineradioMacOS v1.4.0 使用说明

## 安装

1. 从 [Releases](https://github.com/YiIimini/Mineradio-MacOS/releases) 下载对应架构的 `.dmg`：
   - Apple Silicon (M1-M4)：`Mineradio-MacOS-1.4.0-mac-arm64.dmg`
   - Intel：`Mineradio-MacOS-1.4.0-mac-x64.dmg`
2. 打开 `.dmg`，将 MineradioMacOS 拖入 Applications 文件夹
3. **首次打开**：右键（Control+点击）MineradioMacOS → 选择"打开" → 再次点击"打开"（只需一次）

> macOS 提示"无法验证开发者"是正常的，右键打开可绕过。

## Home 页

启动后默认显示 Home 页，包含：

- **左侧卡片**：中国国旗 + 毛选经典语录随机展示
- **我的歌单**：打开左侧歌单库
- **每日推荐**：登录后同步推荐歌曲
- **私人电台**：从推荐和歌单开播
- **继续听**：最近播放记录
- **听歌画像**：播放偏好分析
- **常听歌手**：播放偏好汇总
- **为你准备**：底部横滑推荐卡片

点击 Home 按钮可随时回到此页面。点红旗区域可展开/收起。

## 播放音乐

### 搜索
- **All**：聚合网易云 + QQ 搜索结果
- **NE**：仅网易云
- **QQ**：仅 QQ 音乐
- **LX**：LX 音源搜索（无需登录）
- **Podcast**：播客搜索
- 搜索结果点击即可播放

### 导入本地音乐
- 点击搜索栏右侧 📤 按钮
- 支持 MP3、FLAC、WAV、M4A 格式
- 可同时导入封面图片（JPG/PNG/WebP）

### 歌单
- 登录后左侧面板同步网易云/QQ 歌单
- 支持「我的歌单」和「收藏歌单」
- 歌单内搜索：面板顶部搜索框

## 播放控制

| 操作 | 方式 |
|---|---|
| 播放/暂停 | 底部 ▶ 按钮 / 空格键 |
| 上一首/下一首 | ⏮ ⏭ 按钮 / ← → 键 |
| 音量 | 底部音量图标弹出滑块 / ↑ ↓ 键（±5%） |
| 播放模式 | 🔀 按钮：顺序/随机/单曲循环 |
| 喜欢/收藏 | ♡ 红心 / + 收藏到歌单 |
| 音质 | 底部音质标签：超清母带/高清臻音/无损/极高/标准 |
| 进度 | 拖动进度条 / 点击跳转 |
| 迷你队列 | 📋 按钮查看当前播放列表 |

### 默认播放模式
视觉控制台 → 高级参数 → 默认播放模式：可设置启动时的播放模式（顺序/随机/单曲）。

## 歌词系统

### 舞台歌词（3D）
播放时自动显示大号 3D 歌词，可鼠标拖拽旋转视角。

### 多行歌词
视觉控制台 → 歌词外观 → 显示行数（0-20），≥2 时在 3D 歌词周围显示上下文行。周边行有阶梯层次感和 Z-depth。

### 桌面歌词
视觉控制台 → 叠加效果 → 桌面歌词：全屏置顶浮窗显示歌词。
- 鼠标中键锁定/解锁穿透
- 支持拖动位置
- 帧率可设 24/30/60/120/无上限

### 歌词外观
视觉控制台 → 歌词外观：可调节歌词颜色（自动/手动/封面取色）、字体（12 种）、字间距、行距、字重、大小、位置（XYZ）、角度。

### 自定义歌词
右键播放器 → 自定义歌词：支持 LRC 格式或纯文本，会自动按歌曲时长铺开。

## 3D 歌单架

- 鼠标移到屏幕**右侧边缘**自动呼出
- **滚轮**浏览卡片，**点击**打开详情，**Shift+滚轮**强制滚卡片
- 模式：侧栏（默认）/ 舞台 / 关闭
- 显示设置：自动隐藏 / 常驻
- 内容开关：显示/隐藏播客歌单，合并收藏歌单
- 外观：大小、位置（XYZ）、角度、透明度、背景透明度、主题色

## 视觉控制台

点击右下角 **🎚** 按钮或右侧 FX 悬浮按钮打开。

### 视觉预设（10 种）
| 编号 | 名称 | 说明 |
|---|---|---|
| 0 | emily 封面 | 封面粒子·丝绸 |
| 1 | 滚筒 | 隧道·沉浸感 |
| 2 | 星球 | 星球·雕塑感 |
| 3 | 虚空 | 无粒子·自定义背景 |
| 4 | 唱片 | 唱片·圆形封面 |
| 5 | 星河 | 壁纸粒子·音乐律动 |
| 6 | 安魂 | 骷髅·YUI7W |
| 8 | 泡沫 | 珍珠虹彩·柔波浮沉 |
| 9 | 不规则 | 音律炫酷·不规则柱状 |
| 10 | 棋盘 | 棋盘式海浪·棋盘格律动 |

### 炫酷配色
选择炫酷类预设（8/9/10）后出现配色选择：
- **泡沫/不规则**：暗夜、水墨、紫金、珊瑚、极光
- **棋盘**：极简、水墨、暗夜、霓虹、森林

### 棋盘地面 EQ
选棋盘后展开：网格密度、振幅、运动速度。密度越高柱体越密，振幅控制起伏幅度。

### 主控滑块
律动强度、立体感、封面清晰度、镜头晃动、歌词溢光

### 叠加效果开关
浮空粒子层、电影镜头、歌词溢光、鼓点溢光、歌词光粒、歌词镜头绑定、粒子溢光、轮廓高亮

### 高级参数
- **画质档位**：eco / balanced（默认）/ high / ultra
- **性能后台策略**：自动优化 / 保持运行 / 停止释放
- **直播后台保持**：后台或最小化时保持视觉渲染
- **粒子尺寸、流速、扭曲、色彩张力、溢光强度、离散感、背景压缩**

### 用户存档
可保存/加载视觉配置快照（JSON），含所有参数、颜色、歌词布局。

## 手势控制

**需要摄像头权限**。视觉控制台 → 3D/手势 → 摄像头交互，三选一：

- **关闭**：不使用摄像头
- **手势触碰**（视觉手势）：手掌推开粒子、捏合旋转封面、握拳收束粒子
- **手势控歌**（播放控制）：✊ 拳头→播放、☝ 食指→下一首、✋ 手掌→暂停

手势控歌参数可调：拳头/食指/手掌阈值、保持时间、冷却间隔。点 `?` 查看判别逻辑。

## LX 音源系统

无需登录，社区驱动。搜索栏选「LX」标签页，或在 All 模式下自动兜底。

### 音源管理
点击搜索栏右侧 🌐 按钮打开管理面板：
- **刷新**：重新加载音源目录
- **上传**：上传 .js 音源脚本，自动验证格式
- **启用/禁用**：单独开关音源
- **删除**：移除失败或不需要的音源

音源脚本放在 `音源/音源/` 目录。新电脑首次启动自动从内置目录复制。

> 部分音源依赖网络环境，验证失败可尝试刷新。

## 常见问题

**Q: 封面粒子全是暗紫色？**
A: 正常现象，播放有封面的歌曲后粒子会显示封面颜色。

**Q: 歌词不同步？**
A: 可切换歌词来源（原词/自定义）。右键播放器手动导入 LRC 文件。

**Q: 手势识别不准或太灵敏？**
A: 视觉控制台 → 3D/手势 → 调整阈值滑块。保证光线充足、手在摄像头视野内。

**Q: CPU/GPU 占用高或画面卡顿？**
A: 画质档位调至 eco 或 balanced，关闭浮空粒子层和电影镜头。

**Q: 应用关闭后再打开是黑屏？**
A: v1.4.0 已修复。如仍遇到，右键 Dock 图标 → 退出 → 重新打开。

**Q: 音源上传失败？**
A: 确认脚本包含 `send("inited")` 和 `on("request")` 的 LX 标准格式。网络问题也可能导致验证失败。

**Q: 如何自定义快捷键？**
A: 视觉控制台 → 高级参数 → 热键设置，可修改全局热键。

**Q: 安装后占用多少空间？**
A: 约 270MB（主要为 Electron 运行时）。设置和数据存储在 `~/Library/Application Support/Mineradio-MacOS/`。
